Плагины

Мы собрали список полезных плагинов для различных редакторов кода, которые помогут улучшить вашу продуктивность.

    VS Code

  • Prettier: Форматирование кода с поддержкой множества языков.
  • ESLint: Инструмент для поиска и устранения проблем в JavaScript/TypeScript коде.
  • Live Server: Запускает локальный сервер и автоматически обновляет страницу при изменении файлов.
  • GitLens: Расширяет встроенные функции Git, предоставляет информацию о авторстве строк кода.
  • Python: Полноценная поддержка Python, включая отладку, linting и поддержку Jupyter.
  • Debugger for Chrome: Отладка приложений на JavaScript с использованием Google Chrome.
  • IntelliCode: ИИ-подсказки для автозаполнения кода.
  • Path Intellisense: Автозаполнение путей файлов в проекте.

    Brackets

  • Bracket Highlighter: Подсветка соответствующих скобок и тегов.
  • Emmet: Быстрое написание HTML и CSS через сокращения.
  • Live Preview: Визуализация изменений в реальном времени в браузере.
  • Code Formatting: Автоматическое форматирование кода для улучшения читаемости.
  • Preprocessor: Работа с препроцессорами вроде Sass и LESS.

    Sublime text

  • Package Control: Менеджер пакетов для установки и управления плагинами.v
  • Emmet: Поддержка сниппетов и ускоренное написание HTML и CSS.
  • SublimeLinter: Инструменты для линтинга кода на разных языках.
  • GitGutter: Показ изменений в файлах, отслеживаемых Git, прямо в редакторе.
  • Color Scheme - Default: Позволяет менять цветовые схемы для удобства работы с кодом.
  • Markdown Preview: Превью Markdown-документов.

    Notepad++

  • XML Tools: Инструменты для работы с XML, включая форматирование и валидацию.
  • NppFTP: FTP-клиент для загрузки и скачивания файлов напрямую из Notepad++.
  • JSON Viewer: Удобный просмотр и форматирование JSON-файлов.
  • Compare Plugin: Сравнение двух файлов для выявления различий.
  • Function List: Список функций в файле для быстрого навигации.

    Vim

  • Vim-Plug: Менеджер плагинов для простого управления плагинами.
  • NERDTree: Файловый менеджер для удобного просмотра структуры проекта.
  • fzf: Быстрый поиск файлов и содержимого с помощью fuzzy search.
  • vim-airline: Строка состояния с информацией о текущем состоянии редактора.
  • Auto-pairs: Автоматическое закрытие скобок и кавычек.
  • vim-commentary: Удобные команды для комментирования и раскомментирования кода.
  • ale: Асинхронное линтинг и исправление ошибок.
